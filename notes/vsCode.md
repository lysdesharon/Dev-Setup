Visual Studio Code (VS Code)

Visual Studio Code est un éditeur de code gratuit développé par Microsoft.
 Il permet de créer, modifier et organiser des projets de développement grâce à une interface intuitive, un terminal intégré et un vaste catalogue d'extensions. Il prend en charge de nombreux langages de programmation comme HTML, CSS, JavaScript, Python et bien d'autres, ce qui en fait un outil incontournable pour les développeurs.
